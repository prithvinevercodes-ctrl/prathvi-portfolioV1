## 🚀 About

This portfolio started as a rough layout sketched on paper and gradually evolved into a fully responsive website.

The goal was to build a clean, modern portfolio while learning React, component-based development, responsive design, and deployment workflows.

It will continue evolving as I build more Java Full Stack projects.

## 💻 Getting Started

Clone the repository

```bash
git clone https://github.com/prithvinevercodes-ctrl/prathvi-portfolio-v1.git
```

Go into the project

```bash
cd prathvi-portfolio-v1
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
